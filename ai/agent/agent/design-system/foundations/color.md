## Usage Guidelines

### Step Usage

| Steps | Usage |
|-------|-------|
| 01-02 | App backgrounds |
| 03-04 | Subtle backgrounds, hover states |
| 05-06 | Borders, separators |
| 07-08 | Solid backgrounds (secondary) |
| 09 | Solid backgrounds (primary), interactive |
| 10 | Hovered solid backgrounds |
| 11 | Low-contrast text, secondary text |
| 12 | High-contrast text, primary text |

### Notes

- Steps 01-12 map from lightest to darkest in light mode
- Steps 01-12 map from darkest to lightest in dark mode
- Step 09 is the primary interactive/brand color for each scale
- Overlay uses black with varying alpha for modal backgrounds
